CMSC 197 Mini-Project

## Topic
Classification of Selected Philippine Bird Species Using Acoustic Features and Classical Machine Learning

## Members

- Lin, Yvonne

## Planned Pipeline
1. Collect bird recordings and metadata from Xeno-canto
2. Preprocess audio clips
3. Extract acoustic features (MFCC, spectral features)
4. Train classical ML models
5. Evaluate classification performance
